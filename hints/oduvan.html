O que significa a palavra return?

A palavra reservada return é usada em uma função para especificar o valor que deve ser retornado para quem chama a função. Quando uma função é chamada, ela executa o seu código e pode ou não retornar um valor específico para o código que a chamou.

Por exemplo, podemos escrever no nosso código o comando para imprimir algo na tela com uma função dentro: print(soma). Nosso compilador irá buscar o que a função retornou, ou seja, o que vem depois da palavra reservada return.

O que são os comandos assert?

A função assert é uma ferramenta de depuração em Python que ajuda os desenvolvedores a verificar se uma determinada condição é verdadeira durante a execução de um programa. A função assert é usada para testar se uma condição é verdadeira e, se não for, ela interrompe a execução do programa e lança uma exceção.
